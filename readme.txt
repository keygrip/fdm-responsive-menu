Slide Out Sidebar Menu

== Description ==

**Slide Out Sidebar Menu** Creates a responsive menu based on the SIDR jQuery plugin.  The menu is easily customizable via a settings menu.


= Based on =
* [Sidr](github.com/artberri/sidr) jQuery library

= Website =
https://flyingdonutmedia.com/

== Installation ==

1. Upload the fdm-responsive-menu folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Appearance => Mobile Menu and choose your settings.

== Frequently Asked Questions ==

= How do I display it on my site? =

Use the shortcode [fdm-responsive-menu]
